# PrepAI – AI Interview Analyzer

PrepAI is a full-stack AI-powered interview preparation platform that helps candidates prepare for job interviews by analyzing **Job Descriptions (JD)**, **resumes**, and **candidate self-descriptions**. Using the **Google Gemini API**, it generates personalized interview reports, identifies skill gaps, suggests interview questions, and creates customized preparation plans.

---

## Features

*  AI-powered interview analysis using Google Gemini API
*  Upload resume (PDF) for AI-based resume analysis
*  Analyze Job Description (JD)
*  Candidate self-description analysis
*  JD-Resume Match Score
*  Skill Gap Analysis
*  AI-generated Technical Interview Questions & Answers
*  AI-generated Behavioral Interview Questions & Answers
*  Personalized Interview Preparation Plan
*  JWT Authentication with HTTP-only Cookies
*  Responsive UI with smooth animations
*  Cloud deployment with Vercel & Render
